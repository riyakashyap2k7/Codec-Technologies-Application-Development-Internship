# Codec Technologies Internship Tasks

## Android Application Development Internship

This repository contains the Android application projects developed during my internship at Codec Technologies. The projects focus on solving real-world problems through mobile technology while implementing modern Android development practices, local data storage, analytics, and device hardware integration.

## Projects

### Smart Expense Tracker App

A comprehensive personal finance management application designed to help users efficiently monitor and control their spending habits.

#### Features

* Expense and income management
* Categorized transaction tracking
* Data visualization using interactive charts
* Bill and receipt scanning through OCR
* Export financial reports in PDF and Excel formats
* Secure local data storage using Room Database

#### Technologies Used

* Android Studio
* Java/Kotlin
* Room Database
* MPAndroidChart
* OCR Integration
* PDF & Excel Export Utilities

---

### Health & Fitness Tracker

A health monitoring application that assists users in maintaining a healthy lifestyle through activity tracking, hydration management, and progress analysis.

#### Features

* Real-time step counting using device sensors
* Daily water intake tracking
* Meal planning and monitoring
* Reminder notifications using AlarmManager
* Progress analytics and visualization through charts
* Health statistics tracking over time

#### Technologies Used

* Android Studio
* Java/Kotlin
* Android Sensors API
* AlarmManager
* MPAndroidChart
* Room Database
